# POPs HNR — Beta (demo)

Versão **beta/demonstração** do sistema de leitura e avaliação de POPs do
Hospital Nereu Ramos.

⚠️ **Modo local:** esta versão roda sem servidor — os dados (cadastro, provas,
resultados) ficam salvos **apenas no navegador do aparelho** em que você usar.
Não há sincronização entre dispositivos. É só para demonstrar a experiência.

- **Colaborador:** identifica-se (nome, matrícula, setor), lê os POPs e faz as
  avaliações (questões sorteadas e embaralhadas).
- **Gestor:** acesso com a senha **`gestor123`** (apenas no beta).

A versão real, multiusuário e com painel central do gestor, roda em hospedagem
com PHP + MySQL (ver repositório principal).
